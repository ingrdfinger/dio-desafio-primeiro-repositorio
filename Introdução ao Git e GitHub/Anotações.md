# Anotações :pencil:

### Comandos Básicos

- dir = lista
- cd (change directory) = navega para local específico
- cd .. = volta
- mkdir = cria pasta
- echo = retorna. Por exemplo: echo hello (irá retornar o hello)
- echo hello > hello.txt (exemplo) - criou o arquivo de texto
- del = deletar arquivos
- rmdir (remove directory)
